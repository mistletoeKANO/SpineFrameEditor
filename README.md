# ActionFrameEditor-Spine

#### 介绍
Unity 基于Spine 的帧动画编辑工具

#### 软件架构

本工具 依赖 unity-Spine 动画库, 可以去官网下载最新版.
编辑器使用 UIElement 编写

#### 使用说明

1.ESkeletonAnimation 为 GameObject 基础组件,继承自SkeletonAnimation, 基于当前库的需要做了一些拓展.

2.Tool菜单下可打开 编辑器.

3.具体使用 自行查看
